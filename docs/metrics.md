# 指标口径说明

## 平均年化

- 默认口径：近 5 年复权收盘价 CAGR
- 公式：`(当前复权价 / 起始复权价)^(1/N) - 1`
- 数据不足时降级为 3 年；仍不足返回 `null`
- 数据源：东方财富日 K 线接口 `push2his`

## 股息率

- 优先使用东方财富分红接口中的 `DIVIDENT_RATIO`
- 若字段缺失，降级为：`最近 12 个月每股分红 / 当前价`
- 数据源：`RPT_SHAREBONUS_DET` + 实时行情接口

## 分红率

- 口径：`每股分红 / 基本每股收益(EPS)`
- 计算：`(PRETAX_BONUS_RMB / 10) / BASIC_EPS`
- 若 EPS 缺失，返回 `null` 并附带 notes 提示
